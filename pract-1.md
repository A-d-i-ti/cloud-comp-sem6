Practical 1 — Introduction to Cloud Platforms

### What is Cloud Computing?
Cloud computing is the delivery of computing services — including servers, storage, databases, networking, software, and analytics — over the internet ("the cloud"). Instead of owning physical hardware, users rent computing resources from cloud providers and pay only for what they use.

### Major Cloud Platforms

**1. Amazon Web Services (AWS)**
AWS is the world's largest and most widely used cloud platform, launched by Amazon in 2006. It offers over 200 fully featured services from data centers globally. Key services include EC2 (compute), S3 (storage), VPC (networking), and RDS (database). AWS follows a pay-as-you-go pricing model and offers a Free Tier for 12 months covering services like 750 hours/month of EC2 t2.micro and 5GB of S3 storage.

**2. Microsoft Azure**
Azure is Microsoft's cloud platform launched in 2010. It is particularly popular among enterprises already using Microsoft products like Windows Server, Active Directory, and Office 365. Key services include Azure Virtual Machines (compute), Azure Blob Storage, and Azure Virtual Network. Azure offers a free tier with $200 credits for the first 30 days.

**3. Google Cloud Platform (GCP)**
GCP is Google's cloud platform launched in 2011. It is known for its strengths in data analytics, machine learning, and containerization (Kubernetes originated at Google). Key services include Compute Engine (VMs), Cloud Storage, and VPC. GCP offers $300 free credits for 90 days for new users.

Pricing Calculators
All three platforms provide online pricing calculators to estimate costs before using services:


AWS: AWS Pricing Calculator (calculator.aws)

Azure: Azure Pricing Calculator (azure.microsoft.com/pricing/calculator)

GCP: Google Cloud Pricing Calculator (cloud.google.com/products/calculator)


**These calculators allow users to select services, configure usage parameters, and get a monthly cost estimate — helping avoid unexpected bills.
Benefits of Cloud Computing**

Cost Efficiency — No upfront hardware investment, pay only for usage
Scalability — Scale resources up or down based on demand instantly
Availability — Data centers across the globe ensure high uptime
Security — Enterprise-grade security managed by the provider
Flexibility — Access resources from anywhere via the internet
**
Limitations of Cloud Computing**

Requires stable internet connectivity
Data privacy and compliance concerns
Vendor lock-in risk
Costs can escalate if resources are not monitored properly

Conclusion
AWS, Azure, and GCP are the three dominant cloud platforms, each with unique strengths. AWS leads in market share and service variety, 
Azure excels in enterprise integration, and GCP stands out in data and ML workloads. Understanding their dashboards, key services, 
and pricing models is the first step in working effectively with cloud computing.
